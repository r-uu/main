# Docker Health Check Library
Automatische Gesundheitsprüfungen für Docker-Entwicklungsumgebungen mit Auto-Fix.
## 📦 Features
- **Health Checks** - Prüft Docker Daemon, PostgreSQL, Keycloak, JasperReports
- **Auto-Fix** - Startet Container automatisch, erstellt fehlende Realms
- **Einfache API** - Minimal invasiv, gut wartbar
## 🚀 Verwendung in DashApp
Siehe: `root/app/jeeeraaah/frontend/ui/fx/src/main/java/.../DashApp.java`
Health Checks werden automatisch beim Start ausgeführt.
