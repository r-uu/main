# AI-Prompts für r-uu Projekt

Sammlung wiederverwendbarer AI-Prompt Templates für verschiedene Entwicklungsaufgaben.

## Struktur

- `code-review/` - Prompts für Code-Reviews
- `documentation/` - Prompts für Dokumentationserstellung
- `refactoring/` - Prompts für Refactoring-Aufgaben
- `testing/` - Prompts für Test-Generierung
- `maven/` - Maven-spezifische Prompts

## Verwendung

Kopiere den gewünschten Prompt und passe ihn an deinen spezifischen Kontext an.

### Beispiel: Code Review

```
Ich habe folgenden Code geschrieben:
[DEIN CODE]

Bitte führe ein Code-Review durch mit Fokus auf:
- Maven Best Practices für Multi-Module Projekte
- Java Coding Standards
- Potenzielle Bugs oder Performance-Probleme
```

## Eigene Prompts hinzufügen

Erstelle neue Prompt-Dateien in den entsprechenden Unterordnern.
Verwende aussagekräftige Namen und füge Kommentare mit Verwendungshinweisen hinzu.
